# No Enchantment

**No Enchantment** 是一个 Minecraft NeoForge 模组，旨在彻底移除游戏中的附魔机制。

对于那些希望通过禁用附魔来增加游戏难度、或者打造“低魔/无魔”整合包的作者来说，这是一个理想的解决方案。模组不仅禁止了附魔的生成，还主动将已有的附魔物品转换为普通物品。

## 📦 模组信息 / Mod Info

- **支持版本:** Minecraft 1.21.1 (NeoForge)
- **作者:** Jasdew Starfield
- **协议:** MIT License

## ✨ 主要功能 / Features

这个模组通过多种手段确保游戏中不会出现附魔。所有功能均可自由开关：

1.  **禁止战利品附魔**
    * 从宝箱、遗迹战利品或生物装备中生成的物品将不再带有附魔。
    * 附魔书将变为普通的书。
    * 通过 Mixin 拦截了 `LootPool` 和 `EnchantmentHelper` 的逻辑。

2.  **禁止随机附魔计算**
    * 禁用钓鱼、村民交易或其他依赖随机附魔算法产生的附魔。

3.  **物品主动净化（保底选项）**
    * **附魔书转换**: 所有的“附魔书”在被拾取、掉落或生成时，会自动转换为普通的“书”（保留自定义命名）。
    * **移除附魔**: 任何带有附魔的物品（剑、盔甲等）在被玩家拾取或丢弃时，其附魔属性会被强制移除。

4.  **禁用附魔台**
    * 玩家无法打开附魔台界面。右键点击附魔台将不会有任何反应。

5.  **移除铁砧经验消耗**
    * 作为没有经验修补的补偿，使用铁砧无需消耗经验。这便于玩家随意修复物品。

6.  **JEI 支持**
    * 在 JEI 面板中隐藏附魔书。
    * 隐藏铁砧的附魔配方。
    * 隐藏石磨（Grindstone）的配方分类。

## ⚙️ 配置文件 / Configuration

模组提供了丰富的配置项，你可以在 `config/noenchantment-common.toml` 中进行调整。默认情况下所有功能均为开启状态。
